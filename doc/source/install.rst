Install
*******

Using tourbillon
================


To install tourbillon-celery at a command prompt type: ::

	$ sudo -i tourbillon install tourbillon-celery



Using pip
=========


To install tourbillon-celery using pip enter: ::

	$ pip install tourbillon-celery

.. note::
	
	If you have installed tourbillon in a virtualenv you must install tourbillon-celery in the same virtualenv.
